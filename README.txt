Add the ability to add locations to a map, based on category, in relation to a single location.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `pepperlillie-nearby-locations.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('nearby_locations'); ?>` in your templates or [nearby_locations] on posts or pages
